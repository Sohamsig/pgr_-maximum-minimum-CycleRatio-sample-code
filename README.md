The program computes both the maximum cycle ratio and minimum cycle ratio of a directed weighted graph using Howard’s Cycle Ratio algorithm from the Boost Graph Library.

The algorithm analyzes cycles in the graph and determines the ratios based on the assigned edge weights. The resulting cycle ratios represent the optimal values for the graph according to the algorithm.

For the example graph used in the program, the output produced is:
Maximum Cycle Ratio: 1
Minimum Cycle Ratio: 1

This result indicates that the detected cycle in the graph has the same value for both the maximum and minimum cycle ratio under the given edge weights.

The exact cycle identified may vary depending on the graph structure and weights, but the computed ratio values represent the optimal cycle ratios for the given graph.
